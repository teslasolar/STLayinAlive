About

STLayinAlive is a parametric 3D model library built for Konomi Systems. It features a unique SCADA-style tag provider system AND the revolutionary Konomi System - a multi-agent AI architecture that runs entirely on CPU without any GPU dependency.

All models are defined programmatically using JSCAD and can be customized through simple parameter changes. The tag provider system allows for sophisticated parameter validation, history tracking, and real-time updates.

The Konomi System brings distributed AI intelligence with:

  • eVGPU: CPU-based tensor operations (matmul, conv, activation)
  • FemtoLLM: 16-dim nano AI model (4MB RAM, 0.1s/req)
  • BlockArray: 1000¬≥ sparse compute grid with LLM@coordinates
  • Cube: 9-agent coordination system (8 vertices + 1 central)
  • APIs: REST (port 3001) + WebSocket (port 3002)

Performance: Runs on laptops, scales linearly with CPU cores, processes 1M cubes efficiently, all without requiring expensive GPU hardware.
