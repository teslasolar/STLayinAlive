About

STLayinAlive is a parametric 3D model library built for Konomi Systems. It features a unique SCADA-style tag provider system that enables real-time parameter management, making it perfect for industrial applications and rapid prototyping.

All models are defined programmatically using JSCAD and can be customized through simple parameter changes. The tag provider system allows for sophisticated parameter validation, history tracking, and real-time updates.
